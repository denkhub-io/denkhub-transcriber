Termini di utilizzo

DenkHub Transcriber è un software gratuito sviluppato da Enchelli Dario (denkhub.io), distribuito "così com'è" (as is), senza garanzie di alcun tipo, espresse o implicite.

Privacy e dati

L'applicazione funziona interamente in locale sul tuo computer. Nessun audio, video o testo trascritto viene inviato a server esterni. I tuoi dati non lasciano mai il tuo dispositivo.

Componenti open source

Questo software utilizza componenti open source tra cui whisper.cpp (MIT), FFmpeg (GPL) e Electron (MIT). L'elenco completo delle licenze è disponibile nelle impostazioni dell'app.

Limitazione di responsabilità

L'autore non è responsabile per danni diretti o indiretti derivanti dall'utilizzo del software, incluse imprecisioni nelle trascrizioni. L'utente è responsabile della verifica del contenuto trascritto prima di qualsiasi uso professionale o legale.
